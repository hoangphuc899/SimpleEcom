# SimpleEcom - Website bán hàng trực tuyến
Công nghệ sử dụng: ASP.NET Core 8, Entity Framework Core, SQLite, Bootstrap 5.
Tính năng nổi bật: hỗ trợ quản lý sản phẩm, giỏ hàng, quản lý đơn hàng, hệ thống voucher, ...

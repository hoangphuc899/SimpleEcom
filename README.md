# 🛒 SimpleEcom - ASP.NET Core MVC Project

### 🌟 Giới thiệu
Đây là dự án Website thương mại điện tử cá nhân được xây dựng nhằm mục đích thực hành các kỹ năng lập trình Fullstack với .NET.

### 🛠 Công nghệ sử dụng
- **Backend:** .NET 8, Entity Framework Core, SQLite
- **Frontend:** Bootstrap 5, JQuery, SweetAlert2
- **Khác:** Git, MVC Pattern

### 🚀 Tính năng nổi bật
- **Quản lý sản phẩm:** Xem chi tiết, lọc theo danh mục.
- **Hệ thống giỏ hàng:** Thêm/Xóa sản phẩm, tính toán giá trị đơn hàng.
- **Hệ thống Voucher:** Áp dụng mã giảm giá thông minh.
- **Admin Dashboard:** Quản lý đơn hàng, duyệt báo cáo bình luận vi phạm.
- **Bảo mật:** Phân quyền Admin/User bằng Identity.
